# a11y - Accessibility

WCAG (Web Content Accessibility Guidelines) accessibility tools. Determine the readability of text against background colors, and find the best choice among multiple candidate colors.

## Install

```ts
import xcolor from '@xpyjs/color'
import a11yPlugin from '@xpyjs/color/plugins/a11y'

xcolor.use(a11yPlugin)
```

## Methods

### isReadable(other, level?)

Determine whether the combination of the current color and another color meets WCAG readability standards.

| Parameter | Type | Default | Description |
| --- | --- | --- | --- |
| `other` | `XColorInput` | — | Target color to compare |
| `level` | `WcagLevel` | `'AA'` | WCAG level |

**WcagLevel**: `'AA'` | `'AAA'` | `'AA-large'` | `'AAA-large'`

**Return value**: `boolean`

```ts
xcolor('#000000').isReadable('#ffffff')              // true (contrast ratio 21:1)
xcolor('#000000').isReadable('#ffffff', 'AAA')       // true
xcolor('#777777').isReadable('#888888')              // false (insufficient contrast)
xcolor('#777777').isReadable('#888888', 'AA-large')  // may pass for large text
```

### mostReadable(candidates, options?)

Find the color with the highest contrast ratio against the current color from a list of candidate colors.

| Parameter | Type | Description |
| --- | --- | --- |
| `candidates` | `XColorInput[]` | List of candidate colors |
| `options` | `{ level?: WcagLevel }` | Configuration options |

**Return value**: `XColor`

```ts
xcolor('#333333').mostReadable(['#ffffff', '#eeeeee', '#cccccc']).toHex()
// '#ffffff' — highest contrast ratio
```

### wcagLevel(other)

Return the highest WCAG level met by the two colors.

| Parameter | Type | Description |
| --- | --- | --- |
| `other` | `XColorInput` | Target color to compare |

**Return value**: `string` (e.g. `'AAA'`, `'AA'`, `'AA-large'`, or `'Fail'`)

```ts
xcolor('#000000').wcagLevel('#ffffff')  // 'AAA'
xcolor('#666666').wcagLevel('#999999')  // 'AA-large' or 'Fail'
```

## Practical Scenarios

### Ensure Text Readability

```ts
const bg = xcolor('#1a1a2e')
const textColor = bg.mostReadable(['#ffffff', '#e0e0e0', '#a0a0a0'])
// Automatically selects the text color with the best contrast ratio
```

### Validate Design Mockups

```ts
const pairs = [
  ['#333', '#fff'],
  ['#666', '#eee'],
  ['#999', '#ccc']
]

pairs.forEach(([text, bg]) => {
  const readable = xcolor(text).isReadable(bg)
  const level = xcolor(text).wcagLevel(bg)
  console.log(`${text} on ${bg}: ${readable ? '✓' : '✗'} (${level})`)
})
```
