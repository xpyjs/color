# a11y - 无障碍

WCAG (Web Content Accessibility Guidelines) 无障碍工具。判断文本与背景色的可读性，并在多个候选色中找到最佳选择。

## 安装

```ts
import xcolor from '@xpyjs/color'
import a11yPlugin from '@xpyjs/color/plugins/a11y'

xcolor.use(a11yPlugin)
```

## 方法

### isReadable(other, level?)

判断当前颜色与另一个颜色的组合是否满足 WCAG 可读性标准。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `other` | `XColorInput` | — | 比较目标颜色 |
| `level` | `WcagLevel` | `'AA'` | WCAG 等级 |

**WcagLevel**：`'AA'` | `'AAA'` | `'AA-large'` | `'AAA-large'`

**返回值**：`boolean`

```ts
xcolor('#000000').isReadable('#ffffff')              // true（对比度 21:1）
xcolor('#000000').isReadable('#ffffff', 'AAA')       // true
xcolor('#777777').isReadable('#888888')              // false（对比度不足）
xcolor('#777777').isReadable('#888888', 'AA-large')  // 大字可能通过
```

### mostReadable(candidates, options?)

在候选颜色列表中，找到与当前颜色对比度最高的颜色。

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `candidates` | `XColorInput[]` | 候选颜色列表 |
| `options` | `{ level?: WcagLevel }` | 配置选项 |

**返回值**：`XColor`

```ts
xcolor('#333333').mostReadable(['#ffffff', '#eeeeee', '#cccccc']).toHex()
// '#ffffff' — 对比度最高
```

### wcagLevel(other)

返回两个颜色满足的最高 WCAG 等级。

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `other` | `XColorInput` | 比较目标颜色 |

**返回值**：`string`（如 `'AAA'`、`'AA'`、`'AA-large'`、`'Fail'`）

```ts
xcolor('#000000').wcagLevel('#ffffff')  // 'AAA'
xcolor('#666666').wcagLevel('#999999')  // 'AA-large' 或 'Fail'
```

## 实际场景

### 确保文本可读

```ts
const bg = xcolor('#1a1a2e')
const textColor = bg.mostReadable(['#ffffff', '#e0e0e0', '#a0a0a0'])
// 自动选择对比度最佳的文本颜色
```

### 检查设计稿

```ts
const pairs = [
  ['#333', '#fff'],
  ['#666', '#eee'],
  ['#999', '#ccc']
]

pairs.forEach(([text, bg]) => {
  const readable = xcolor(text).isReadable(bg)
  const level = xcolor(text).wcagLevel(bg)
  console.log(`${text} on ${bg}: ${readable ? '✓' : '✗'} (${level})`)
})
```
