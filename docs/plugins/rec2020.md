# rec2020

Rec. 2020 广色域色彩空间支持。Rec. 2020 是为 UHD/4K/8K 电视和 HDR 内容定义的色彩空间标准。

## 安装

```ts
import xcolor from '@xpyjs/color'
import rec2020Plugin from '@xpyjs/color/plugins/rec2020'

xcolor.use(rec2020Plugin)
```

## 方法

### toRec2020()

返回 Rec. 2020 对象。各通道值 0–1。

```ts
xcolor('#ff0000').toRec2020()
// { r: 0.79, g: 0.23, b: 0.07 }
```

### toRec2020String()

返回 `color(rec2020 ...)` 格式字符串。

```ts
xcolor('#ff0000').toRec2020String()
// 'color(rec2020 0.79 0.23 0.07)'
```

## 输入解析

```ts
xcolor('color(rec2020 0.79 0.23 0.07)').toHex()
```

## 格式化

```ts
xcolor('#ff0000').toString('rec2020')  // 'color(rec2020 0.79 0.23 0.07)'
```
